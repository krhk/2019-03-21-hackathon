mattLib has URL rewrite, you can specify routes in routes.php

$routes = [
"home" => "common/home"
];

You can also specify URL variables :

$routes = [
"home/:message" => "common/home"
];

Using URL variables :

Your message is: <?= $_url[':message'] ?>

You can make these variables non-optional by adding * :

$routes = [
"home/:message*" => "common/home"
];
